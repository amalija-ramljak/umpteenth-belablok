# Test and deploy
### No build
Scan QR code and open link in NativeScript Preview app
```bash
ns preview
```

### Deploy app for testing
USB or WiFi debugging
```bash
ns run [platform] # platform: [android, ios]
```

### Build app
```bash
ns build [platform] # platform: [android, ios]
```
